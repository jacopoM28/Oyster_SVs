# Oyster_SVs
Folder containing scripts for SVs analyses in oysters

**1.Within-individuals_SVs:** Scripts for identification of heterozygous SVs in oysters reference genomes.  
**2.Simulations:** Scripts for benchmarking heterozygous SVs detection in a reference genome.  
**3.Polymorphic_TEs:** Scripts for identification of mobile element insertions in the *C. ariakensis* WGS population genomic dataset.  
**4.Between-Individuals_SVs:** Scripts for SVs detection in the *C. ariakensis* WGS population genomic dataset.  
**6.Additional_Scripts:** Additional python scripts.
